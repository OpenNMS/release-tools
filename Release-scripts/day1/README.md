# Day 1 tools
This tool is designed to help with Day 1 activities.

## How to use the tool
1) Clone repository
2) Update `configuration/credential.conf`
3) Run tool by providing the version you are working on and the path to `changelog.adoc` file
 a) Run `python3 main.py -v 35.0.0 -c "/tmp/opennms/docs/modules/releasenotes/pages/changelog.adoc"`
